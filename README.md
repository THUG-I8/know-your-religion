# تطبيق Rabbo

هذا مشروع Flutter بسيط يسمى "Rabbo App".

## المميزات

- تطبيق Flutter أساسي
- واجهة مستخدم بسيطة
- عداد تفاعلي

## كيفية التشغيل

1. تأكد من تثبيت Flutter على جهازك
2. قم بتشغيل الأمر التالي:
   ```bash
   flutter pub get
   flutter run
   ```
